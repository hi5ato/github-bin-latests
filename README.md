# github-bin-latests

This repository provides scripts to easily install the latest versions of `lazygit` and `ghq` on Linux.

## Prerequisites

The following tools are required to be installed on your system:

- `curl`
- `jq`
- `tar`
- `gzip`
- `unzip`

## Installation

### lazygit

```bash
curl -fsSL "https://raw.githubusercontent.com/hi5ato/github-bin-latests/main/install-lazygit.bash" | bash
```

### ghq

```bash
curl -fsSL "https://raw.githubusercontent.com/hi5ato/github-bin-latests/main/install-ghq.bash" | bash
```
